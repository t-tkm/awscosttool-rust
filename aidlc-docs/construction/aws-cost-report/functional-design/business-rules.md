# Business Rules

## BR-01: 集計期間
- 開始日は常に当月1日（実行日の月の1日）
- 終了日は常に当日（実行日）
- 表示上の終了日は「終了日 - 1日」（Cost Explorer の End は当日 0:00 を指すため）

## BR-02: コストフィルタリング
- クレジット適用後: フィルタなし（全 RECORD_TYPE を含む）
- クレジット適用前: RECORD_TYPE ディメンションで "Credit" を Not フィルタで除外

## BR-03: 最小表示閾値
- 0.01 USD 未満のサービスは表示しない
- 除外されたサービスは DEBUG レベルでログに記録する

## BR-04: 合計コスト計算
- API レスポンスの `Total` フィールドが存在する場合はそれを使用
- `Total` が空の場合は `Groups` の各 Amount を合算する
- 負の Amount は 0 として扱う（クレジット等の影響を防ぐ）

## BR-05: 出力順序
- クレジット適用後レポートを先に出力
- クレジット適用前レポートを後に出力

## BR-06: エラー処理
- AWS API 呼び出し失敗時はエラーメッセージを stderr に出力してプロセスを非ゼロ終了コードで終了
- パニックは使用しない（`Result` 型でエラーを伝播）

## BR-07: リージョン
- AWS リージョンは `us-east-1` 固定
- Cost Explorer は us-east-1 エンドポイントのみ対応のため変更不要
